Secure Software Development

Secure Software Development was my third module for the University of Essex Online MSc Computer Science course. I started this 12-week module on 7th November 2023.

Click on the link in the 'Description & Link' column to navigate to the respective piece of work.

Unit Work

Name Unit(s) Description & Link
Collaborative Discussion 1 1, 2, 3 Select an identified coding weakness and create a UML diagram showing the steps leading to the vulnerability.
Blog Post 2 Select five terms from ISO/IEC 27000 Section 3 and write a 300 word blog post on how people can be managed to overcome cyber security attacks from the inside.
Team Discussion 3 Answer questions on the subject of secure programming languages and discuss those answers within our team.
(Codio) Python Tools and Features 3 Codio exercise exploring some of the Python features which improve secure software development.
(Codio) Producer-Consumer Mechanism 4 Codio exercise on the producer-consumer mechanism.
Evil Regex 4 Answer questions on regular expressions (regex) and regular expression denial of service (ReDoS) attacks.
Cyclomatic Complexity 5 Opinion-based exploration of the relevance of cyclomatic complexity today.
(Codio) Testing Linters in Python 6 Codio exercise on the producer-consumer mechanism.
(Codio) Creating a Simple Python Shell 7 Creating my own simple Python shell.
Cryptography Programming Exercise 8 Creating a simple Python program which encrypts a file and outputs the result to another file.
Collaborative Discussion 2 8, 9, 10 Read about the TrueCrypt software and present an ontology design of the identified weaknesses.
(Codio) Developing an API for a Distributed Environment 9 Create a RESTful API which can be used to create and delete user records.
