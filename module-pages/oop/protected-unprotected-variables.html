Protected and Unprotected Variables

Task

In Unit 1, we were tasked with developing a Python program including the use of protected and unprotected variables, a subject which had featured in our reading for this week. In addition to this, for Unit 2 we were asked to create a Python program which included employee-related functionality, such as tracking employee details and allowing an employee to book a day of annual leave.

My program allows storage of employee name, employee number, salary and amount of annual leave within an 'Employee' object. This class also contains methods to change the salary (as salary is a protected variable), return the monthly pay of an employee and book a given number of days of annual leave. In the program, I included some example code for interacting with the 'Employee' object, including a requirement for user input when selecting how many days of annual leave to book. The code for my program can be seen below.

Code

class Employee:
    """Represents an employee.

    attributes: name, employee number, salary, annual leave
    """

    def __init__(self, name, employeeNum, salary, annualLeave=25):
        self.name = name
        self.employeeNum = employeeNum
        self._salary = salary
        self._annualLeave = annualLeave

    def change_salary(self, newSal):
        self._salary = newSal

    def monthly_pay(self):
        return self._salary / 12
        
    def book_holiday(self, holidayLength):
        if holidayLength > self._annualLeave:
            print("You do not have enough annual leave remaining.")
        else:
            self._annualLeave = self._annualLeave - holidayLength
            print("Leave booked!")


e1 = Employee("John Smith", 215180452, 25000, 25)
print(e1._salary)
e1._salary = 30000
print(e1._salary)
print(f"You have {e1._annualLeave} days holiday remaining.")
holiday = int(input("Please enter the number of days to book: "))
e1.book_holiday(holiday)
print(f"You have {e1._annualLeave} days holiday remaining.")
